# Lý thuyết
    Props

- React Elements
    - Sử dụng props giống như với attribute của thẻ html
    - 2 props class, for =>  className, htmlFor
    - Phải tuân theo quy ước có sẵn
- React components
    - Sử dụng props giống như đối số cho function
    - Tự do đặt trên Props
        + Đặt theo camelCase
        + * Có thể bao gồm dấu "-" (gạch ngang)
- Chú ý
    - Props "key" là props đặc biệt
    - Props cơ bản là đối số của Component
        => Props có thể bất cứ kiểu dữ liệu gì
    - Sử dụng Destructuring

  function PostItem(props) { -- Component
    return (
      <div className="post-list"> -- React components
        <div className="post-item">
          <img src={props.image} alt=""></img>

        />
        <PostItem  -- React Component

      </div>
    )
  }
  const root = document.getElementById('root');
  ReactDOM.render(App(), root)
