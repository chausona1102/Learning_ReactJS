Các kiến thức cần nắm trước khi học useEffect
    + Events: add/ remove event listener
    + Observer pattern: Subcribe / unsubcribe
    + Closure
    + Timers: setTimeout, setTimeInterval, clearTimeout, clearInterval
    + useState
    + Mounted / Unmounted
    + ===
    + Call API

     - Chỉ gọi lại callback 1 lần sau khi component mounted
     - Ngăn chặn request trên trình duyệt
     useEffect(callback, [deps]) => 2 tham số: callback và dependencies
     - Callback sẽ được gọi lại mỗi khi deps thay đổi
